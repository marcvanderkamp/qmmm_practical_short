The MSYS Command Line

To run this practical you will need to use the MSYS command line. The MSYS command line is a window in which you can type commands. You can start the command line by clicking the Start button | All Programs | msys | MSYS 1.0.11.

MSYS provides a command line environment that is very similar to that found in UNIX. The following commands are useful;

  • ls : list the contents of the current directory
  • cd : change directory - change into a new directory.
  • mkdir : make a new directory.
  • rmdir : remove an empty directory.
  • rm : remove a file.
  • cp : copy a file.
  • mv : move a file or directory.
  • write : open a file for editing (using windows write)

  • Note that MSys provides useful 'tab completion' - start typing a command, and, if the remainder of the command is not ambiguous, hitting the Tab key will complete the typing for you.
  • Note also that MSys has a command history - if you hit the upward (or downward) arrow, you can view previous commands, edit them if needed, and enter them again.
