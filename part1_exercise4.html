Part 1: QM/MM Simulations

Exercise 4 - The Gas-Phase Reaction with QM Arg90

The final simulation to run is the calculate the reaction energies for going from chorismate to prephenate in the gas-phase, while bound to a solitary Arg90 residue (modelled using QM).

The input script is qm_r90qm.inp. You will need to modify it to set the R variable in the beginning to do the calculation for the values of the reaction coordinate that correspond to the reactant (-1.8), transition state (-0.3) and product (1.8). Do this by editing the file by typing write qm_r90qm.inp and saving three different files, one for each of the three values of the reaction coordinate.

For each of the three files, run a CHARMM simulation. Do this by typing; 

./charmm < FILENAME.inp > FILENAME.log
where FILENAME is the name of the file, e.g. if you have named the file for the reactant qm_r90qm_-1.8.inp, then you would need to type; 
./charmm < qm_r90qm_-1.8.inp > qm_r90qm_-1.8.log
